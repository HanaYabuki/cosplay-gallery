# Cosplay Gallery

是花枝丸做来专门存放自己的 Cosplay 照片的 Web 项目！

> 布局其实是抄的某个本子网站wwww

## 使用方式
1. 先搞一个cosplay-db项目
2. 对每个相册按照所示结构进行信息整理
3. 部署到静态网站服务上
